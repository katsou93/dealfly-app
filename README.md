# ✈️ DealFly – Smarter Flugdeal-Scanner

Günstigste Flüge aus NRW – kein Login, direkt zur Buchung.

## Setup
1. API-Key: https://tequila.kiwi.com/register
2. npm install && cp .env.example .env.local
3. TEQUILA_API_KEY in .env.local eintragen
4. npm run dev → http://localhost:3000

## Deploy auf Vercel
TEQUILA_API_KEY als Environment Variable setzen → Deploy klicken.
